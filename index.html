Access the menu easily by scanning the QR Code

This feature is used to scan the QR code on the table, then a menu will be displayed at each restaurant. There are other features in this feature, namely flash, rotate, and close.
• Flash to compensate for lack of brightness when shooting in low light situations such as indoors or night views,
• Rotate will rotate about one of the current axes of the camera's coordinate system,
• Close to return to the previous menu.

Find the restaurant supported and nearby

This feature is used to find the nearest restaurant and supports the use of Monday app. After determining the restaurant, we can see the details of the restaurant.

Select a payment method

This feature is used to display order payment bills, then you can choose the desired payment method, which is divided into 2, namely cash and non-cash payments.
